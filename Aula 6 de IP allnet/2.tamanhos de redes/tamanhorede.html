Diferenças entre WAN, MAN e LAN

O que é uma Rede LAN?

LAN significa Local Area Network (Rede de Área Local). É uma rede que conecta dispositivos em uma área limitada, como uma casa, escola ou escritório. Exemplos:

  • Rede de computadores em uma empresa.
  • Wi-Fi doméstico.

O que é uma Rede MAN?

MAN significa Metropolitan Area Network (Rede de Área Metropolitana). É uma rede que cobre uma área maior que uma LAN, como uma cidade ou campus universitário. Exemplos:

  • Rede de câmeras de segurança em uma cidade.
  • Rede de universidades interligadas em uma região.

O que é uma Rede WAN?

WAN significa Wide Area Network (Rede de Área Ampla). É uma rede que cobre grandes distâncias, conectando cidades, países ou até continentes. Exemplos:

  • Internet.
  • Rede de uma empresa com filiais em diferentes países.

Diferenças principais

Tipo de Rede Área de Cobertura Exemplo
LAN Pequena (casa, escritório) Wi-Fi doméstico
MAN Média (cidade, campus) Rede de câmeras de segurança
WAN Grande (países, continentes) Internet
