O que são as Portas na Rede?

Entendendo com uma analogia

Imagine que seu computador é como um prédio com vários apartamentos. Cada apartamento tem uma função específica — um para e-mail, outro para navegador, outro para jogos online. As portas são os números que identificam cada um desses apartamentos para que os dados cheguem no lugar certo.

O que são Portas?

Portas são números usados pelos protocolos para direcionar os dados para o aplicativo correto dentro do computador. Elas vão de 0 a 65535.

  • Portas bem conhecidas (0–1023): usadas por serviços populares como HTTP, HTTPS, FTP, etc.
  • Portas registradas (1024–49151): usadas por aplicativos específicos de empresas.
  • Portas dinâmicas ou privadas (49152–65535): usadas temporariamente por programas e sistemas operacionais.

Exemplos de Portas Comuns

Porta Serviço Função
80 HTTP Sites normais (sem criptografia)
443 HTTPS Sites seguros (com criptografia)
25 SMTP Envio de e-mails
21 FTP Transferência de arquivos
53 DNS Tradução de endereços (nomes para IPs)

Atividade prática no computador

  1. Abra o Prompt de Comando (cmd).
  2. Digite: netstat -an e pressione Enter.
  3. Observe as portas usadas nas conexões ativas.
  4. Anote 3 portas abertas e pesquise para que servem.
