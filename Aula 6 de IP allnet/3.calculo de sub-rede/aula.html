Aula de Redes - Máscara de Sub-rede e CIDR

1. O que é Máscara de Sub-rede?

Ela define qual parte do endereço IP pertence à rede e qual parte pertence ao host (computador, impressora, etc). Exemplo:

IP: 192.168.1.10
Máscara: 255.255.255.0

2. O que é CIDR?

CIDR significa Classless Inter-Domain Routing. É uma forma abreviada de escrever a máscara de sub-rede. Exemplo:

255.255.255.0 = /24

Como calcular o número de hosts?

Use a fórmula: 2ⁿ - 2, onde n é o número de bits de host.

4. Sobre o /32

O /32 representa um único IP fixo. Ele não permite host, rede ou broadcast. Usado em:

  • Loopback: 127.0.0.1/32
  • Regras de firewall ou roteamento
